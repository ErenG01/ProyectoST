INTRUCCIONES DE INSTALACION
------------------------------------------------------------

Para instalar el sitio, sigue estos pasos:


-crear una base de datos llamada "servitec" en PHPMYADMIN MYSQL.

-cargar en PHPMYADMIN en docuemnto llamado "servitec.sql"

-descargar o cargar el proyecto de servitec en github

-alojar la carpeta descargada en C:\xampp\htdocs.

-ejecutar desde el navegador http://localhost/ProyectoST


USO
-------------------------------------------------------------
para saber como funciona cada parte de la apgina web te recomendamos leer el manuel "Proyecto_ST"
